Pack portfolio bilingue FR/EN

Fichiers inclus
- index.html
- translations.js

Installation
1. Place index.html et translations.js dans le même dossier.
2. Garde ton dossier images/ inchangé.
3. Ouvre index.html dans le navigateur ou déploie-le sur ton hébergement.

Fonctionnement
- Le bouton FR/EN est intégré dans le menu.
- La langue choisie est mémorisée via localStorage.
- Les textes traduisibles sont centralisés dans translations.js.

Limite actuelle
- Ce pack ne contient pas les images, car elles n'étaient pas présentes dans les fichiers fournis ici.
- Si tu veux, je peux ensuite te générer une V2 avec davantage de textes externalisés ou une structure encore plus modulaire.
